## Overview

This comprehensive guide covers the complete installation and configuration of WordPress on Windows using a modern, professional stack: PHP 8.x, MariaDB, and Caddy web server. This setup provides excellent performance, security, and maintainability for both development and production environments.

### What You'll Build

- **WordPress CMS**: Latest version with professional configuration
- **PHP 8.x**: Modern PHP runtime with optimized settings
- **MariaDB**: High-performance MySQL-compatible database
- **Caddy Web Server**: Automatic HTTPS and reverse proxy
- **NSSM Service Manager**: Professional Windows service management

### Architecture Overview

```
Internet ‚Üí Caddy (Port 80/443) ‚Üí PHP-FPM (Port 9000) ‚Üí WordPress ‚Üí MariaDB (Port 3306)
```

### Key Benefits

- **Professional Setup**: Enterprise-ready configuration
- **Automatic HTTPS**: SSL certificates managed by Caddy
- **High Performance**: Optimized PHP and database settings
- **Security Hardened**: Best practices throughout
- **Service Management**: Automatic startup and monitoring
- **Scalable**: Ready for production workloads

## Prerequisites

Before beginning the installation, ensure your system meets all requirements:

### System Requirements

#### Minimum Requirements
- **Operating System**: Windows 10 64-bit (version 1903) or Windows 11
- **RAM**: 4GB system memory (8GB+ recommended)
- **Storage**: 10GB available disk space (SSD recommended)
- **Processor**: 64-bit processor (2+ cores recommended)

#### Recommended Requirements
- **Operating System**: Windows 11 or Windows 10 Pro (latest version)
- **RAM**: 16GB+ system memory
- **Storage**: SSD with 50GB+ available space
- **Processor**: Multi-core 64-bit processor (4+ cores)
- **Network**: Stable internet connection

### Software Prerequisites

- **Administrator Access**: Required for service installation
- **PowerShell 5.1+**: For automation scripts
- **Visual C++ Redistributable**: For PHP and MariaDB
- **Domain Name**: For production deployment (optional for development)

### Network Requirements

- **Ports 80 and 443**: Available for web traffic
- **Port 3306**: Available for database (can be changed)
- **Port 9000**: Available for PHP-FPM
- **Firewall**: Configured to allow necessary traffic

## Component Downloads and Verification

### Step 1: Download Required Components

#### PHP 8.x (Latest Stable)
Download PHP 8.x (Dynamic - Always Latest)
```powershell
# Get the latest PHP 8.x version dynamically
$phpReleasePage = Invoke-WebRequest -Uri "https://windows.php.net/downloads/releases/archives/" -UseBasicParsing
$latestPhp8 = ($phpReleasePage.Links | Where-Object { $_.href -match "php-8\.\d+\.\d+-nts-Win32-vs16-x64\.zip$" } | Sort-Object href -Descending | Select-Object -First 1).href
$phpUrl = "https://windows.php.net/downloads/releases/archives/$latestPhp8"
$phpVersion = ($latestPhp8 -split "-")[1]
$phpOutput = "$env:TEMP\php-$phpVersion-nts-Win32-vs16-x64.zip"
Write-Host "Downloading PHP version: $phpVersion"
Invoke-WebRequest -Uri $phpUrl -OutFile $phpOutput
```

**Alternative - Direct Latest Release:**
```powershell
# For the absolute latest stable release (may include newer major versions)
$phpUrl = "https://windows.php.net/downloads/releases/latest/php-nts-Win32-vs16-x64.zip"
$phpOutput = "$env:TEMP\php-latest-nts-Win32-vs16-x64.zip"
Invoke-WebRequest -Uri $phpUrl -OutFile $phpOutput
```
Verify download
```
Get-FileHash $phpOutput -Algorithm SHA256
```

**Manual Download**: [PHP for Windows](https://windows.php.net/download/)
- Choose **Non Thread Safe (NTS)** version
- Select **x64** architecture
- Download **ZIP** package

#### NSSM (Service Manager)

Download NSSM (Dynamic - Always Latest)

```powershell
# Get the latest NSSM version dynamically
$nssmPage = Invoke-WebRequest -Uri "https://nssm.cc/download" -UseBasicParsing
$latestNssmLink = ($nssmPage.Links | Where-Object { $_.href -match "nssm-\d+\.\d+\.zip$" } | Select-Object -First 1).href
$nssmUrl = "https://nssm.cc$latestNssmLink"
$nssmVersion = ($latestNssmLink -split "/")[-1] -replace "\.zip$", ""
$nssmOutput = "$env:TEMP\$nssmVersion.zip"
Write-Host "Downloading NSSM version: $nssmVersion"
Invoke-WebRequest -Uri $nssmUrl -OutFile $nssmOutput
```

**Alternative - Direct Latest:**
```powershell
# NSSM doesn't change frequently, but this ensures latest
$nssmUrl = "https://nssm.cc/release/nssm-2.24.zip"  # Fallback to known stable
$nssmOutput = "$env:TEMP\nssm-latest.zip"
try {
    $nssmPage = Invoke-WebRequest -Uri "https://nssm.cc/download" -UseBasicParsing
    $latestLink = ($nssmPage.Links | Where-Object { $_.href -match "nssm-\d+\.\d+\.zip$" } | Select-Object -First 1).href
    if ($latestLink) { $nssmUrl = "https://nssm.cc$latestLink" }
} catch { Write-Host "Using fallback NSSM URL" }
Invoke-WebRequest -Uri $nssmUrl -OutFile $nssmOutput
```

### Step 3: Install and Configure MariaDB

#### Install MariaDB

Install MariaDB silently (Dynamic file detection)

```powershell
# Find the downloaded MariaDB installer dynamically
$mariaInstaller = Get-ChildItem "$env:TEMP\mariadb-*-winx64.msi" | Select-Object -First 1 -ExpandProperty FullName
if (-not $mariaInstaller) {
    Write-Error "MariaDB installer not found in $env:TEMP"
    exit 1
}
Write-Host "Installing MariaDB from: $(Split-Path $mariaInstaller -Leaf)"

$installArgs = @(
    "/i", $mariaInstaller,
    "/quiet",
    "SERVICENAME=MariaDB",
    "PASSWORD=SecureRootPassword123!",
    "UTF8=1"
)

Start-Process -FilePath "msiexec.exe" -ArgumentList $installArgs -Wait -NoNewWindow
Write-Host "‚úÖ MariaDB installed"
```

#### Create PHP-FPM Service

```powershell
# Install PHP as a service
& "C:\Tools\NSSM\nssm.exe" install PHP-FPM "C:\Tools\PHP\php-cgi.exe"
& "C:\Tools\NSSM\nssm.exe" set PHP-FPM Parameters "-b 127.0.0.1:9000"
& "C:\Tools\NSSM\nssm.exe" set PHP-FPM AppDirectory "C:\Tools\PHP"
& "C:\Tools\NSSM\nssm.exe" set PHP-FPM DisplayName "PHP FastCGI Process Manager"
& "C:\Tools\NSSM\nssm.exe" set PHP-FPM Description "PHP FastCGI Process Manager for WordPress"

# Configure service recovery
& "C:\Tools\NSSM\nssm.exe" set PHP-FPM AppExit Default Restart
& "C:\Tools\NSSM\nssm.exe" set PHP-FPM AppRestartDelay 5000

# Start PHP-FPM service
& "C:\Tools\NSSM\nssm.exe" start PHP-FPM
Write-Host "‚úÖ PHP-FPM service created and started"
```
